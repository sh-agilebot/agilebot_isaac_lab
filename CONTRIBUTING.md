# Contributing Guide

Thanks for your interest in contributing!

## What’s in this repo

- `pick_place_demo/`: a runnable pick-and-place demo built on Isaac Lab.
- `assets/`: Agilebot-related asset configuration.

## Development setup (recommended)

1. Install NVIDIA Omniverse Isaac Sim and Isaac Lab (matching versions).
2. Activate your Isaac Lab conda environment (for example `isaaclab`).
3. Make sure Agilebot USD assets are configured via `assets/agilebot.py`.

## Pull requests

- Keep PRs focused (one feature/fix per PR).
- Include a short description, motivation, and test evidence (screenshots/logs are fine if you can’t run full tests).
- If you add CLI flags, document them in the relevant README.

## Issues

When filing a bug, please include:

- OS + GPU + driver version
- Isaac Sim version + Isaac Lab version
- The exact command you ran and the full error log
